# Smart Parking System - Complete Setup Guide

## Overview
This is a complete Smart Parking System with three components:
1. **picker.py** - Setup script to mark parking slots
2. **backend.py** - Real-time detection and monitoring
3. **React Dashboard** - Web UI for visualization

## Prerequisites

### System Requirements
- Python 3.8+
- Node.js 16+ (for React dashboard)
- OpenCV compatible system

### Python Dependencies
\`\`\`bash
pip install opencv-python ultralytics easyocr
\`\`\`

### Models (Auto-downloaded)
- YOLOv8n - ~6.3 MB (auto-downloaded on first run)
- EasyOCR - ~100 MB (auto-downloaded on first run)

## Setup Instructions

### Step 1: Prepare Your Parking Lot Image
1. Take a clear photo of your parking lot and save as `parking_lot_img.png`
2. Place it in the same directory as the scripts

### Step 2: Run Picker Script
\`\`\`bash
python scripts/picker.py
\`\`\`

**Instructions:**
- Left-Click and drag to draw rectangles around each parking slot
- Right-Click to remove the last rectangle
- Press 'S' to save (creates `slot_coordinates.pickle`)
- Press 'ESC' to exit without saving

### Step 3: Setup IP Webcam (Mobile)
1. Install IP Webcam app on your phone (Android: Google Play, iOS: App Store)
2. Open the app and start the server
3. Note the camera URL (usually `http://192.168.X.X:8080/video`)
4. Update the URL in `backend.py`:
   \`\`\`python
   CAMERA_URL = "http://192.168.X.X:8080/video"  # Replace with your IP
   \`\`\`

### Step 4: Run Backend Script

\`\`\`bash
.venv\Scripts\activate.bat
python run_all.py
\`\`\`

This will:
- Connect to your IP webcam
- Detect cars in real-time
- Recognize license plates
- Save data to `parking_data.json`
- Display live visualization

### Step 5: Run React Dashboard
\`\`\`bash
npm run dev
\`\`\`

Visit: http://localhost:3000

The dashboard will:
- Auto-refresh every 2 seconds
- Show real-time occupancy status
- Display license plates and billing
- Support Dark/Cappuccino themes

## Troubleshooting

### Camera Not Connecting
- Ensure phone and computer are on same WiFi
- Test URL in browser: `http://192.168.X.X:8080/shot.jpg`
- Check firewall settings

### License Plate Recognition Issues
- Ensure license plates are clearly visible
- Good lighting is essential
- The system generates random plates if OCR fails

### Performance Issues
- Reduce frame resolution in backend.py
- Use YOLOv8n (already the lightest model)
- Run on a machine with GPU support

## Billing Logic
- Base fee: ₹50.00
- Per-second charge: ₹1.00
- Example: 1 hour parking = ₹50 + ₹3600 = ₹3650
